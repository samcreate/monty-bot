# monty.wine

> A bot the wine on the mind.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i monty.wine --save-dev
```

## Usage
```sh
~/ngrok http --hostname bottletalk.ngrok.io 3000
```
Then run

```sh
npm run dev
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
